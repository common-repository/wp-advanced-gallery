Enhanced gallery for WordPress with the following: Magnific Popup, Isotope / Masonry, Owl-Carousel

== Description ==

Enhanced gallery for WordPress. Plugin extends default WordPress gallery by adding some extra options. You can change Animation on hover (Zoom In / Zoom Out), On Hover style (light or dark), choose which info should be displayed along with captions, etc.

**How to use**

Just create a standard WordPress gallery and select gallery options. See screenshots for more information.

**Features:**
- Magnific Popup
- Isotope / Masonry with filters
- Owl-Carousel
- Hover styles and animations
- Custom gallery captions



== Installation ==


1. Upload the plugin files to the '/wp-content/plugins/wp-advanced-gallery' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
